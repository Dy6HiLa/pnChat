# pnChat

pnChat - плагин для Minecraft-сервера, который делает чат понятнее и удобнее.
Он разделяет обычные сообщения рядом с игроком, глобальные сообщения для всего
сервера и отдельный чат для администрации.

## Что делает плагин

- Обычные сообщения видят только игроки рядом.
- Сообщения с `!` в начале видит весь сервер.
- Администраторы могут включить отдельный админ-чат.
- Игроков можно упоминать через `@ник` или просто написав ник онлайн-игрока.
- При упоминании игрок получает звуковое уведомление.
- Чат можно красиво оформить: префиксы, цвета, локальный и глобальный формат.
- Плагин блокирует спам, капс, слишком длинные сообщения, запрещённые слова и
  нежелательные ссылки.
- Можно разрешить только нужные ссылки, например сайт сервера или Discord.
- Если на сервере есть LuckPerms, плагин может показывать префиксы игроков.
- Если игрок замьючен через Essentials, LiteBans, AdvancedBan или CMI, pnChat
  не даст ему писать в чат.

## Как пользоваться

Обычное сообщение отправляется в локальный чат. Его увидят игроки рядом:

```text
Привет
```

Чтобы написать всему серверу, поставьте `!` перед сообщением:

```text
!Всем привет
```

Чтобы упомянуть игрока, напишите его ник через `@`:

```text
@Steve подойди сюда
```

Можно написать ник и без `@`. Если игрок сейчас онлайн, pnChat сам превратит
ник в упоминание:

```text
Привет Steve
```

Если такие автоупоминания не нужны, их можно выключить в `config.yml`.

## Команды

```text
/pnchat reload
```

Перезагружает настройки плагина без перезапуска сервера.

```text
/pnchat ac
```

Включает или выключает админ-чат для вас.

```text
/pnchat adminchat
```

То же самое, что `/pnchat ac`.

## Права

```text
pnchat.admin
```

Даёт доступ к командам `/pnchat reload`, `/pnchat ac`, админ-чату и уведомлениям
об обновлениях. По умолчанию право есть у операторов сервера.

```text
pnchat.color
```

Разрешает использовать цветовые коды в чате, если в настройках включено
требование права на цвета.

## Установка

1. Скачайте или соберите `pnChat-1.0.0.jar`.
2. Переместите JAR-файл в папку `plugins` вашего Paper-сервера.
3. Перезапустите сервер.
4. После запуска откройте папку `plugins/pnChat`.
5. Настройте файлы под свой сервер.
6. Выполните `/pnchat reload` или ещё раз перезапустите сервер.

## Настройка

Главный файл настроек:

```text
plugins/pnChat/config.yml
```

В нём можно изменить:

- радиус локального чата;
- формат локального, глобального и админ-чата;
- звук при упоминании игрока;
- автоупоминания по нику без `@`;
- права для цветного чата;
- максимальную длину сообщения;
- защиту от спама, капса и повторяющихся символов;
- фильтрацию ссылок.

Сообщения, которые видят игроки, находятся здесь:

```text
plugins/pnChat/messages.yml
```

Разрешённые ссылки находятся здесь:

```text
plugins/pnChat/allowed-links.txt
```

Запрещённые слова находятся здесь:

```text
plugins/pnChat/blocked-words.txt
```

После изменения этих файлов используйте:

```text
/pnchat reload
```

## Настройка автоупоминаний

```yaml
plain-name-mentions-enabled: true
plain-name-mention-min-length: 3
plain-name-mention-ignored-words: []
```

`plain-name-mentions-enabled` включает или выключает превращение обычного ника
онлайн-игрока в упоминание.

`plain-name-mention-min-length` задаёт минимальную длину ника для
автоупоминания. Значение `3` подходит для Minecraft-ников.

`plain-name-mention-ignored-words` нужен для исключений. Если ник совпадает с
частым словом на сервере, добавьте его в список:

```yaml
plain-name-mention-ignored-words:
  - admin
  - help
```

## Примеры форматов

```yaml
local-format: '&8[&aL&8] &7{player} &8» &f{message}'
global-format: '&8[&dG&8] &f{player} &8» &f{message}'
admin-format: '&8[&cADMIN&8] &f{player}&7: &f{message}'
```

`{player}` - ник игрока.

`{message}` - текст сообщения.

## Сборка из исходников

Если нужно собрать плагин вручную:

```powershell
.\gradlew.bat build
```

Готовый файл появится здесь:

```text
build/libs/pnChat-1.0.0.jar
```

Для сборки нужна Java 21.
