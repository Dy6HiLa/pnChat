# pnChat

pnChat - плагин для чата на Minecraft-сервере. Он делает общение удобнее: обычные сообщения видят только игроки рядом, важные сообщения можно отправить всему серверу, а у администрации есть отдельный чат.

Плагин рассчитан на обычный сервер: поставил файл в `plugins`, перезапустил сервер и настроил поведение чата под себя.

## Как работает обычный чат

Когда игрок просто пишет сообщение, pnChat отправляет его не всему серверу, а только игрокам рядом.

Радиус задаётся в настройках. Например, если радиус стоит `100`, сообщение увидят только те, кто находится не дальше 100 блоков от игрока. Игроки в другом мире или далеко от отправителя сообщение не увидят.

Такой чат удобен для выживания, ролевых серверов и любых проектов, где не нужно, чтобы каждую локальную переписку видел весь онлайн.

## Как написать всему серверу

Чтобы сообщение увидели все игроки, нужно поставить `!` в начале.

Игрок пишет:

```text
!Всем привет
```

В чат уйдёт глобальное сообщение для всего сервера. Сам знак `!` нужен только для выбора глобального чата.

## Как работает админ-чат

Админ-чат нужен для общения администрации прямо в игре.

Администратор пишет:

```text
/pnchat ac
```

После этого его обычные сообщения будут видеть только игроки с правом `pnchat.admin`. Повторная команда выключает админ-чат.

Это удобно, когда модераторы или админы хотят быстро обсудить ситуацию, не выходя из игры и не показывая сообщения игрокам.

## Как работают упоминания

pnChat умеет упоминать игроков в сообщениях.

Можно написать ник через `@`:

```text
@Steve подойди сюда
```

Можно написать ник без `@`:

```text
Steve подойди сюда
```

Если игрок с таким ником сейчас онлайн, pnChat выделит его ник как упоминание и проиграет ему звук. Регистр не важен: `Steve`, `steve` и `STEVE` будут считаться одним и тем же ником.

Если автоупоминания без `@` мешают, их можно выключить в настройках.

## Защита чата

pnChat проверяет сообщения перед отправкой.

Плагин может блокировать:

- спам;
- слишком много CAPS LOCK;
- слишком длинные сообщения;
- повторяющиеся символы;
- запрещённые слова;
- нежелательные ссылки.

Разрешённые ссылки указываются отдельно. По умолчанию список пустой, поэтому обычные ссылки вроде `www.youtube.com` будут блокироваться. Если какой-то домен на сервере можно писать в чат, добавьте его в `allowed-links.txt`.

## Поддержка других плагинов

pnChat может работать вместе с другими популярными плагинами.

Если установлен LuckPerms, pnChat может показывать донатный префикс игрока прямо в строке чата. В стандартном формате он стоит после `[L]` и `[G]`, чтобы сначала было видно тип чата, а потом группу игрока.

Если установлен PlaceholderAPI, в формате чата можно использовать свои плейсхолдеры. Например, можно добавить мир игрока, баланс, клан или другой текст из плагинов, которые поддерживает PlaceholderAPI.

Если игрок замьючен через Essentials, LiteBans, AdvancedBan или CMI, pnChat не даст ему писать в чат.

Если этих плагинов нет, pnChat всё равно будет работать.

Каждое доставленное локальное, глобальное или административное сообщение также выводится в консоль сервера ровно один раз. Заблокированные модерацией сообщения в консоль не попадают.

## Команды

```text
/pnchat reload
```

Перезагружает настройки без перезапуска сервера.

```text
/pnchat ac
```

Включает или выключает админ-чат для вас.

```text
/pnchat adminchat
```

То же самое, что `/pnchat ac`.

```text
/chatcolor <игрок> <цвет|none>
```

Устанавливает постоянный цвет сообщений игрока. Доступны только русские названия: `черный`, `темно-синий`, `темно-зеленый`, `темно-бирюзовый`, `темно-красный`, `темно-фиолетовый`, `золотой`, `серый`, `темно-серый`, `синий`, `зеленый`, `голубой`, `красный`, `розовый`, `желтый`, `белый`. Также поддерживается HEX, например `#42a5f5`. `none` сбрасывает цвет.

Старый порядок `/chatcolor <цвет> <игрок>` сохранён для совместимости.

```text
/chatfont <игрок> <стиль...|none>
```

Устанавливает постоянные стили: `жирный`, `курсив`, `подчеркнутый`, `зачеркнутый`, `магический`. Можно передать несколько стилей через пробел или запятую, например `/chatfont Steve жирный курсив`. `none` сбрасывает все стили. Старый порядок со стилями перед игроком также поддерживается.

```text
/nick <игрок> <новый ник|none>
```

Меняет отображаемый ник только в сообщениях pnChat. Можно использовать цвета и пробелы. Настройка хранится по UUID и работает для ранее заходивших офлайн-игроков. Если Essentials занял команду `/nick`, используйте `/pnick` или `/pnchat:nick`.

```text
/prefix <игрок> <префикс|none>
```

Устанавливает собственный префикс pnChat. Он заменяет префикс LuckPerms; после `none` снова используется LuckPerms. Для гарантированного вызова доступен псевдоним `/pprefix`.

## Права

```text
pnchat.admin
```

Доступ к командам плагина, админ-чату и уведомлениям об обновлениях.

```text
pnchat.color
```

Разрешение на цветной чат, если в настройках включена проверка права на цвета.

```text
pnchat.chatcolor
pnchat.chatfont
pnchat.nick
pnchat.prefix
```

Доступ к командам управления цветом, стилями, ником и префиксом других игроков.

## Установка

1. Скачайте готовый файл `pnChat-1.0.3.jar` со страницы релиза.
2. Откройте папку вашего сервера.
3. Перенесите `pnChat-1.0.3.jar` в папку `plugins`.
4. Перезапустите сервер.
5. После запуска появится папка `plugins/pnChat`.
6. Настройте чат под свой сервер.
7. Если меняли настройки на включённом сервере, используйте `/pnchat reload`.

## Что можно настроить

Все основные настройки лежат в папке:

```text
plugins/pnChat
```

Главные файлы:

```text
config.yml
messages.yml
allowed-links.txt
blocked-words.txt
chat-styles.yml
```

В `config.yml` настраивается поведение чата: радиус локального чата, формат сообщений, звук упоминаний, цвета, антиспам и фильтр ссылок.

В `messages.yml` можно изменить текст сообщений, которые видят игроки.

В `allowed-links.txt` добавляются ссылки, которые разрешено отправлять в чат. Если файл пустой, ссылки запрещены. Строки с `#` считаются комментариями и не разрешают домены.

В `blocked-words.txt` добавляются слова и фразы, которые нужно блокировать.

В `chat-styles.yml` pnChat автоматически хранит назначенные игрокам цвета, стили, ники и префиксы. Редактировать этот файл вручную обычно не требуется.

## Анонимная статистика

В pnChat встроены `pnLibrary` и bStats с ID проекта `32607`. bStats отправляет только анонимную техническую статистику; отключить её можно глобально в `plugins/bStats/config.yml`.

## Формат сообщений

В настройках можно изменить внешний вид локального, глобального и админ-чата.

```yaml
local-format: '&8[&aL&8] %luckperms_prefix%&7{player} &8» &f{message}'
global-format: '&8[&dG&8] %luckperms_prefix%&f{player} &8» &f{message}'
admin-format: '&8[&cADMIN&8] %luckperms_prefix%&f{player}&7: &f{message}'
```

`{player}` - назначенный через `/nick` ник либо настоящее имя игрока.

`{real_name}` - настоящее имя аккаунта без замены.

`{prefix}`, `{luckperms_prefix}` и `%luckperms_prefix%` - назначенный через `/prefix` префикс либо префикс LuckPerms.

`{message}` - текст сообщения.

Префикс можно поставить куда угодно. Например, если хотите видеть донат после знака локального или глобального чата, оставьте `%luckperms_prefix%` сразу после `[L]` и `[G]`.

В эти строки можно добавлять и другие плейсхолдеры PlaceholderAPI. Главное, чтобы нужный плейсхолдер был установлен и работал на сервере.

Цвета пишутся через обычные Minecraft-коды с `&`.
